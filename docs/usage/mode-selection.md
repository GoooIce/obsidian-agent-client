# 模式选择

某些 agent 支持不同的操作模式，这些模式会改变 agent 的行为方式。

## 什么是模式？

模式是预定义的配置，用于改变 agent 在特定任务中的行为。例如：

- **默认模式**：通用辅助
- **计划模式**：在实现之前专注于规划和架构

## 更改模式

1. 打开聊天面板
2. 在输入框下方寻找 **模式下拉菜单**
3. 从可用选项中选择所需的模式

<p align="center">
  <img src="/images/mode-selection.webp" alt="模式选择下拉菜单" width="400" />
</p>

::: tip
可用模式取决于当前活动的 agent。并非所有 agent 都支持多种模式。
:::

## 模式持久性

选择的模式在当前会话中保持有效。开始新聊天会重置为默认模式。
