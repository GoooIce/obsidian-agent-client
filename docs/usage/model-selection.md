# 模型选择

在不同的 AI 模型之间切换，以优化速度、能力或成本。

## 什么是模型？

模型是具有不同能力的 AI 不同版本：

- **更大的模型**：能力更强，推理更好，更慢，成本更高
- **更小的模型**：响应更快，成本更低，适合简单任务

## 更改模型

1. 打开聊天面板
2. 在输入框下方寻找 **模型下拉菜单**
3. 从可用选项中选择所需的模型

::: tip

:::

## 模型持久性

选择的模型在当前会话中保持有效。你的偏好会在会话之间被记住。

## 成本考虑

使用 API 密钥时，不同模型有不同的定价。请查看提供商的定价页面了解当前费率。
